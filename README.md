# 🏠 Family Utility Tracker

A simple and modern web app to track household utility expenses and visualize spending trends.

## 🚀 Features

* Monthly bill tracking
* Auto calculation (total, average, peak)
* Interactive charts (Chart.js)
* Local storage (no backend needed)
* Clean modern UI (Tailwind CSS)

## 📦 How to Use

1. Open `index.html` in your browser
2. Add your monthly bills
3. View stats and charts instantly
